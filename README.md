# Qt_touch_interface

用Qt做一个界面，用来打开各种软件，实现界面的显示和各个软件的功能

本来我是打算跟着网上的Qt做项目的，但是看到太杂了，于是直接打算把它们全部糅合到一起，做成这样一个项目，秉承着开源的思想，我发布到github，新手做得可能很难看，哈哈哈哈

部分素材来源于网络，侵权联系我删除

# 项目内容

+ App
  + 串口助手
  + 音乐播放器
  + 绘图预览
  + 自定义控件
  + 人脸识别
+ 按钮
  + 关闭
  + 隐藏


# 项目配置

+ Qt版本：6.5.3
  + qmake编译
  + MinGW 11.2.0 64-bit
  + Libraries
    + Qt Mutimedia
    + Qt Serial Port
+ opencv版本：4.9.0
